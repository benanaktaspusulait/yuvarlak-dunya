# Pompom Hills - Character Reference Guide v1.0

---

## Purpose

This guide provides production-ready character references for AI image and AI video generation. It ensures visual consistency across all generated content by providing exact specifications for body shapes, colours (with hex codes), outfits, expressions, and reusable prompts for every angle and pose.

Two naming versions are provided:
- **Version A**: Role-based names (Main Child, Animal Friend, Calm Guide Character)
- **Version B**: Named trio (Kiko, Mimi, Opa)

All visual specifications are identical between versions — only the names change.

---

## Visual Style Constants

These rules apply to ALL character prompts and ALL generated images:

| Rule | Detail |
|------|--------|
| **Style** | Soft 3D cartoon animation |
| **Shape language** | Rounded, circular, no sharp edges |
| **Colour palette** | Soft pastel, warm tones |
| **Lighting** | Warm daylight, gentle shadows |
| **Texture** | Soft matte surfaces, toy-like |
| **Background** | Simple, uncluttered, soft |
| **Feel** | Soft toys come to life |

### Universal Prompt Prefix

```
Soft 3D cartoon animation style, rounded shapes, pastel colours, warm daylight, gentle shadows, toy-like appearance, soft matte surfaces, Pompom Hills world
```

### Universal Avoid List

```
no sharp edges, no dark shadows, no realistic textures, no complex backgrounds, no scary elements, no harsh lighting, no neon colours, no busy compositions, no angular shapes, no straight lines in nature, no metallic surfaces, no glass effects, no photographic elements
```

# CHARACTER 1: MAIN CHILD

---

## 1. Character Overview

| Attribute | Detail |
|-----------|--------|
| **Role name** | Main Child |
| **Species** | Human girl |
| **Age (personality)** | 3–4 years |
| **Main colour** | Coral Pink (#F8BBD0) |
| **Secondary colour** | White (#FFFFFF) |
| **Catchphrase** | "What's that?" |

## 2. Role in the Series

The Main Child is the protagonist and audience surrogate. She asks questions, discovers new things, and models curiosity and emotional growth. She initiates every adventure and leads the trio's exploration of Pompom Hills. The audience sees the world through her eyes.

## 3. Personality

- **Curious**: Always asks "What's that?"
- **Gentle**: Never rough, always soft
- **Kind**: Helps friends, shares everything
- **Brave**: Tries new things (with support)
- **Patient**: Learns to wait (with help)

## 4. Emotional Function for Toddlers

The Main Child mirrors the toddler's own emotional journey. She models:
- Curiosity as a positive trait
- Asking for help when stuck
- Sharing without losing
- Trying new things even when scared
- Feeling sad and recovering safely

## 5. Exact Body Shape

- **Overall**: Snowman-like with arms — two stacked soft circles (head + body)
- **Head-to-body ratio**: Head 40%, Body 60% of total height
- **Head**: Perfectly round, width 1.2× body width
- **No visible neck** — head merges directly into body
- **Arms**: Short, chubby, no elbows visible, simple round shape
- **Hands**: Round, no fingers defined
- **Legs**: Short, chubby, no knees visible
- **Feet**: Round, no toes defined
- **Proportions**: 2.5 heads tall total

## 6. Face Shape
