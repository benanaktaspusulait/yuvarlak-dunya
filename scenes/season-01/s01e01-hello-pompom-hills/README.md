# S01E01 - Hello Pompom Hills v2.1

## Episode Purpose

İlk bölüm, Pompom Hills dünyasını ve ana karakterleri çok basit bir sırayla tanıtır. Çocuk izleyici önce Kiko ile bağ kurar, sonra Mimi'nin güvenli arkadaş rolünü görür.
