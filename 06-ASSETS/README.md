# 06-ASSETS

Genel referans görselleri ve üretim çıktısı olmayan yardımcı görseller için ayrılmıştır.

## Ne buraya konur?
- Karakter/mekan dışı genel referans görseller (mood board, stil örnekleri, paletler)
- Geçici veya araştırma amaçlı görseller

## Ne buraya KONMAZ?
- **Mekan (world) görselleri** → `environment/` (kanonik konum)
- **Karakter çizimleri** → `01-CHARACTERS/drawings/`
- **Prop görselleri** → `03-PROPS/`
- **Marka / kanal görselleri** → `07-BRANDING/`
- **Render edilmiş videolar** → `videos/`
