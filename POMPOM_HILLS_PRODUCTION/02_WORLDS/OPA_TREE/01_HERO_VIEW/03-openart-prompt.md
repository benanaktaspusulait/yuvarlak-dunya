# Opa's Tree — OpenArt World Builder Prompt

> Use this short plain-text prompt in the OpenArt World Builder text field.

---

## If uploading 1 image

Use the image as the primary reference and paste this prompt into the text field:

```
A large, ancient Pompom Tree with a thick warm brown trunk and a wide, round, lush green canopy. One broad horizontal branch extends outward, forming a natural seat. Dappled sunlight filters through the leaves creating soft light patterns on the grass below. The tree stands alone in a gentle clearing. Soft green grass surrounds the base. Warm, wise, protective atmosphere. Children's storybook style, soft matte handcrafted look. No other trees nearby, no modern objects, no sharp edges, no dark shadows.
```

## If uploading 2–4 images

Do not rely on the text field — it will be disabled. Instead:

- Image 1 (front): wide shot of the tree from below
- Image 2 (back): view from the branch seat looking down
- Image 3 (left): side view showing the branch structure
- Image 4 (right, optional): close-up of the seat branch

All four images must share the same lighting direction, sky color, and green palette.

---

## After the world is built

Use the bible at `00_CANON/` as your production reference when framing shots, placing characters, and running QA inside the generated world.
