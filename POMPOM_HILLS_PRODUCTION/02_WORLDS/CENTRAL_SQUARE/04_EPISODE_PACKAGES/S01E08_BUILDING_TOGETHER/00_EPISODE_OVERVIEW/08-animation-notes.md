# S01E08 — Animation Notes

> All motion slow, soft, preschool-safe. Tower stays ≤ head level. The fall (Shot 04) is soft and gentle.

---

## Noah (scale 100)

| Action | Notes |
|---|---|
| Arrange blocks | Gentle, curious handling (Shot 01) |
| Show flat block | Calm demonstration (Shot 02) |
| Careful pointing | Hand + eyes + head + body aligned to target block (Shot 06) |
| Place blocks | Slow, deliberate stacking (Shots 03, 07) |
| Reassure Arda | Warm, calm supportive gesture (Shot 05) |
| Proud smile | Bright, happy resolution (Shot 08) |

---

## Arda (scale ~85, slightly smaller/younger)

| Action | Notes |
|---|---|
| Arrive | Small steps from a rounded path (Shot 01) |
| Lift a block | Curious question gesture (Shot 02) |
| Steady the tower | Both hands holding while Noah places (Shot 07) |
| Shoulder drop | Small, brief discouraged feeling (Shot 05) |
| Follow the point | Turns toward the block Noah indicates (Shot 06) |
| Beam / celebrate | Happy, relieved (Shot 08) |
