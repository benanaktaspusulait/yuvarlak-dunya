## Naming Convention

```
[category]-[description]-[date]-[status].png
```

Examples:
- `canon-exterior-hero-view-2026-07-05-approved.png`
- `test-exterior-low-angle-2026-07-05-review.png`
- `variation-exterior-sunset-2026-07-05-archive.png`

## Status Tags

- `approved` — Canonical reference, passes all QA checks
- `review` — Needs review against QA Checklist
- `reject` — Failed QA, kept for reference only
- `archive` — Historical or alternative versions
