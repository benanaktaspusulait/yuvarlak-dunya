# S01E13 — Animation Notes

---

## Animation Principle

This episode should feel like real preschool animation.

Avoid demonstration-style acting.
No character should perform a lecture.
The discovery should emerge through small actions, pauses and shared looking.

Add a natural micro-action every 3-5 seconds.

---

## Luca

| Action | Notes |
|---|---|
| Climbing | Small energetic steps uphill, curious but not rushed |
| Pausing | Stops when he notices the size of the clouds |
| Looking up | Head tilt, blinking, quiet wonder |
| Pointing | Small uncertain gesture toward the moving cloud |
| Thinking | Looks from cloud to Mimi, then back to the sky |
| Waiting | Holds one hand out gently while listening |
| First drop reaction | Notices his hand quietly before speaking |
| Smiling | Warm, soft discovery smile |

---

## Mimi

| Action | Notes |
|---|---|
| Walking | Gentle, observant steps beside Luca |
| Looking up | Notices details before speaking |
| Pointing | Small calm point toward one cloud |
| Listening | Lets Luca's question sit for a moment |
| Observing | Watches the cloud, grass and sky instead of explaining |
| Waiting | Listens with Luca instead of answering quickly |
| Small nod | A soft realization, not teacher confidence |
| Smiling | Shared joy when the rain begins |

---

## Environment

| Element | Motion |
|---|---|
| Clouds | Shot 01 normal, Shot 02 moving, Shot 03 fuller, Shot 04 slightly darker, Shot 05 first drop, Shot 06 gentle light rain |
| Grass | Soft wind movement, visible during quiet beats |
| CH-STONE-01 | Stable existing Cloud Hill visual anchor near the children |
| Sky | Diffused cloudy light remains consistent |
| Waiting air | Wind becomes briefly softer before the drop |
| First raindrop | One clean drop lands on Luca's hand |
| Rain | Gentle vertical drops begin slowly, then continue softly |

---

## Micro-Action Rhythm

| Time | Micro-Action |
|---|---|
| 0-5 sec | Footsteps and grass movement |
| 5-10 sec | Luca stops and looks up |
| 10-15 sec | Mimi follows Luca's gaze |
| 15-20 sec | Mimi points to one cloud |
| 20-25 sec | Luca watches the cloud move |
| 25-30 sec | Wind lifts the grass |
| 30-35 sec | Both crouch or settle near the grass |
| 35-40 sec | Silent thinking beat |
| 40-45 sec | Both look back up at the fuller cloud |
| 45-50 sec | The wind softens |
| 50-55 sec | Luca slowly holds out one hand |
| 55-60 sec | Mimi listens and watches the cloud |
| 60-65 sec | First drop lands on Luca's hand |
| 65-70 sec | Luca notices the drop quietly |
| 70-75 sec | Mimi looks from the drop to the cloud |
| 75-80 sec | Gentle rain begins |
| 80-85 sec | Luca and Mimi share a quiet smile |
| 85-90 sec | Both look back at the clouds as rain continues softly |

---

## Avoid

- Mimi explaining with big hand gestures.
- Teacher-like nodding or pointing.
- Overacting surprise.
- Fast rain start.
- Dramatic storm animation.
- Camera-driven emotion replacing character behavior.
