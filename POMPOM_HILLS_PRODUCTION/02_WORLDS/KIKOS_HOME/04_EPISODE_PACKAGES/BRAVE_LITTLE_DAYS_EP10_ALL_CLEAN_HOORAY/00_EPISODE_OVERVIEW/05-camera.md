# EP10 — Camera Plan

## Camera Identity

| Property | Value |
|---|---|
| Lens Range | 28–35 mm |
| Height | Child eye level (~1.0 m virtual) |
| Movement Style | Gentle, stable. No handheld shake. |
| Perspective | Slightly above toys, at character eye level |
| Depth of Field | Moderate — characters sharp, background slightly soft |

## 8-Shot Camera Table

## Camera Continuity Rules

1. **Kiko always screen-left**, Mimi always screen-right (except shot 07 wide).
2. **Lens stays 28–35mm** — never telephoto, never ultra-wide.
3. **Eye level always** — camera never looks down on children.
4. **No zoom** — all focal changes are cuts, not zooms.
5. **Pan only in shots 01 and 07** — wide establishing shots.
6. **Static for all medium shots** — stable, calm feeling.
7. **Shot 05 = Fresh Quality Reset Anchor** — verify clean start, correct lens, correct exposure.
