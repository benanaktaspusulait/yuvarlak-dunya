## Forbidden Sounds

| Sound | Reason |
|---|---|
| Background music | Rule: NO background music |
| On-screen text sounds | Rule: NO on-screen text |
| Harsh clattering | Would feel like chaos, not satisfaction |
| Scolding tone | Violates safe-exposure |
| Timer / countdown | Would create urgency — opposite of calm tidying |
| Cartoon boing / slapstick | Too silly for the warm tone |
| Adult voices | No adult characters |
| Door slamming | Too aggressive |
| Vacuum / cleaning machines | Too industrial for preschool calm |

## Sound Design Principles

1. **Toy sounds are satisfying** — soft, gentle, rewarding. Each placement should feel like a small celebration.
2. **Room tone is warm** — never cold or clinical. The playroom should feel alive but calm.
3. **Silence is allowed** — gaps between dialogue are fine. Don't fill every second.
4. **Progress = sound** — as toys find their homes, the room sounds cleaner. Less visual noise = less sonic noise.
