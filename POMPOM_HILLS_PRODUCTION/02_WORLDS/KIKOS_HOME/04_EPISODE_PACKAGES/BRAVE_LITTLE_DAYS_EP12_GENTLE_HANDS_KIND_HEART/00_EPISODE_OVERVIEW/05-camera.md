# EP12 — Camera Plan

## Episode: "Gentle Hands, Kind Heart"

---

## GLOBAL CAMERA RULES
- **Lens range:** 28–35mm equivalent
- **Height:** Child eye level (~80–100cm from ground)
- **Movement:** Slow, gentle — no fast pans or zooms
- **Style:** Observational, intimate, as if kneeling beside the characters
- **Focus:** Soft backgrounds, sharp subjects

---

## SHOT-BY-SHOT CAMERA

### Shot 01 — "A Little Butterfly"
| Property | Value |
|----------|-------|
| Lens | 35mm |
| Position | Wide establishing, child eye level |
| Movement | Slow push-in from wide to medium-wide |
| Subject | Kiko + butterfly on flower |
| Focus | Deep (both Kiko and flower visible) |
| Framing | Kiko screen-left, flower+butterfly screen-right |
| Duration | 15s |
| Notes | Establishes space. Warm garden light. Butterfly is small but noticeable. |

| Property | Value |

## CAMERA MOVEMENT MAP
```
Shot 01: PUSH-IN (establishing → focus)
Shot 02: STATIC/SLIGHT FOLLOW (eagerness)
Shot 03: STATIC (coaching stability)
Shot 04: MICRO PUSH-IN (suspense)
Shot 05: PULL-BACK + RACK FOCUS (magic reveal) ★
Shot 06: STATIC/FLOAT (warmth)
Shot 07: STATIC (intimacy)
Shot 08: TILT UP (release, freedom)
```

## LENS SEQUENCE
```
35mm → 30mm → 32mm → 28mm → 30mm → 32mm → 28mm → 35mm
Wide → Med → Med → Close → Close → Med → Close → Wide
```
Pattern: Opens wide, narrows to intimacy, returns to wide for resolution.
