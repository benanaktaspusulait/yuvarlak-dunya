# EP12 — Animation Notes

## Episode: "Gentle Hands, Kind Heart"

---

## GLOBAL ANIMATION RULES

| Rule | Value |
|------|-------|
| Frame rate | 24fps target |
| Character motion | Soft, rounded, no sharp movements |
| Butterfly motion | Delicate, floating, wing-focused |
| Background motion | Minimal — swaying foliage, light shifts |
| Camera motion | Slow, gentle, child eye level |
| Lip sync | Only speaking character; listener mouth closed |

---

## SHOT-BY-SHOT ANIMATION

### Shot 01 — "A Little Butterfly"

| Element | Animation |
|---------|-----------|
| Kiko | Walking → pause → eyes track butterfly. Natural child gait, bare feet |
| Butterfly | Flutters in from screen-right, gentle arc, lands on flower |
| Flower | Slight sway as butterfly lands. Subtle weight response |
| Foliage | Gentle breeze movement — slow, rhythmic |
| Light | Warm dappled shifting, soft shadows on ground |
| **Key pose** | Kiko mid-step, head turning to follow butterfly |

### Shot 02 — "I Want to Touch It"

| Element | Animation |
|---------|-----------|
| Kiko | Leans forward quickly, hands reaching out. Eager but not rough |
| Kiko's hands | Open palms, fingers spread — reaching too fast |
| Kiko's face | Eyes wide, mouth open with excitement. Eyebrows raised |
| Butterfly | Stays on flower, slight wing flutter — notices movement |
| **Key pose** | Kiko leaning forward, hands reaching, face lit with eagerness |
| **Notes** | Movement speed: EAGER (not ROUGH). She reaches too fast, not too hard |

### Shot 03 — "Soft and Gentle"

| Element | Animation |
|---------|-----------|
| Mimi | Steps in screen-right, soft paw gesture — calming, open |
| Mimi's ears | Slight droop of calmness, then return to neutral |
| Kiko | Pauses mid-reach, pulls back slightly, looks at Mimi |
| Kiko's hands | Slowly lower from eager reach to relaxed position |
| Butterfly | Stays on flower, unbothered |
| **Key pose** | Mimi's open-paw gesture, Kiko's recalibration moment |
| **Notes** | Mimi is coaching, not scolding. Warm body language throughout |

### Shot 04 — "One Gentle Finger"

| Element | Animation |
|---------|-----------|
| Kiko's finger | Extends slowly — one finger, deliberate, careful |
| Kiko's face | Concentrated, eyebrows slightly furrowed with focus |
| Butterfly | Tilts head, watches finger approach. Wings adjust slightly |
| Flower | Still, as if holding its breath |
| **Key pose** | The gap between fingertip and butterfly — maximum tension |
| **Notes** | EXTREMELY slow movement. The gap closing is the entire animation story |

### Shot 05 — "It Lands on Me" ★ FRESH ANCHOR

| Element | Animation |
|---------|-----------|
| Butterfly | Lifts from flower, brief hover, lands softly on fingertip |
| Kiko's face | Eyes widen. Mouth small 'o'. Pure amazement |
| Kiko's body | Freezes completely — holds perfectly still |
| Kiko's eyes | Slight moisture/glisten of wonder (not tears) |
| Wings | Gentle, slow flutter while on finger |
| **Key pose** | Butterfly on fingertip, Kiko's amazed eyes |
| **Notes** | HIGHEST PRODUCTION VALUE. Magical. Time stops. Smooth, graceful motion |

### Shot 06 — "So Beautiful"

| Element | Animation |
|---------|-----------|
| Mimi | Warm smile, slight bounce of joy, proud expression |
| Kiko | Still, transfixed. Barely breathes |
| Butterfly | Rests calmly on finger. Occasional wing micro-movement |
| Light | Warm golden glow — connection moment |
| **Key pose** | Three characters together — Kiko, Mimi, butterfly |
| **Notes** | Warmth of connection. All three in harmony |

### Shot 07 — "Soft and Gentle" (whispered)

| Element | Animation |
|---------|-----------|
| Kiko | Profile view, lips move to whisper. Eyes soft |
| Butterfly | Rests on finger in foreground. Barely moves |
| Background | Soft bokeh, garden blur |
| **Key pose** | Kiko's profile with butterfly — intimate, peaceful |
| **Notes** | Most intimate shot. Minimal animation. Stillness is beauty |

### Shot 08 — "Gentle Hands, Kind Heart"

| Element | Animation |
|---------|-----------|
| Butterfly | Lifts off, graceful upward arc, flies away |
| Kiko | Watches butterfly go, then peaceful smile |
| Mimi | Smiles beside Kiko |
| Light | Warm, golden — fading slightly as butterfly departs |
| **Key pose** | Kiko and Mimi watching butterfly fly into soft sky |
| **Notes** | Butterfly departure is beautiful, not sad. Eyes open on final frame |

---

## CHARACTER ANIMATION PRIORITY

1. **Butterfly** — Most animation frames. Flutter, hover, land, rest, fly
2. **Kiko** — Full range: walking, reaching, concentrating, amazed, peaceful
3. **Mimi** — Fewer poses: coaching gesture, warm smile, proud observation
4. **Garden** — Subtle: sway, light shifts, breeze

---

## TRANSITION ANIMATION

| Shot pair | Transition style |
|-----------|-----------------|
| 01→02 | Cut on Kiko's gaze shift |
| 02→03 | Cut as Mimi enters frame |
| 03→04 | Cut to close-up of finger |
| 04→05 | CONTINUOUS — butterfly lifts, lands (no cut feel) |
| 05→06 | Cut to wider view after landing |
| 06→07 | Cut to intimate profile |
| 07→08 | Cut to wide as butterfly lifts off |

Shot 04: Concentration (micro-movements, held breath)
Shot 05: Amazement (frozen, eyes wide) ★
Shot 06: Warmth (still, connected, gentle smiles)
Shot 07: Peace (whisper, minimal movement)
Shot 08: Understanding (watching, then smile)
