| Element | Sound | Notes |

## Forbidden Sounds

| Sound | Reason |
|-------|--------|
| Background music | CRITICAL RULE — no music |
| On-screen text sounds | No text displayed |
| Loud impacts | No harsh sounds |
| Sharp edges | No metallic, glass, or hard sounds |
| Traffic / outdoor | Interior scene only |
| Electronic sounds | No tech, no phones |
| Animal calls | Characters are stylized, not realistic |
| Crowd noise | Only 2-3 characters |
| Laughter track | No canned laughter |
| Dramatic stings | No dramatic cues |

## Sound Mixing Notes

| Parameter | Value |
|-----------|-------|
| Voice level | Primary — clear and soft |
| Ambience level | Low — supportive, not dominant |
| SFX level | Soft — gentle, child-friendly |
| Dynamic range | Narrow — no loud/soft extremes |
| Reverb | None — dry, intimate interior |
| Master level | Moderate — safe for children |
