### Production Rules

1. **No background music** — voice and ambience only
2. **No on-screen text** — dialogue is spoken, not displayed
3. **Child eye level** — all shots at 80-100cm height
4. **28-35mm focal length** — warm, intimate framing
5. **Warm daytime lighting** — natural, soft, cozy
6. **Simple vocabulary** — preschool-appropriate dialogue
7. **Safe exposure** — no scolding, mess is normal, teamwork is joyful
8. **Opa minimal** — wise owl speaks 1-2 lines per appearance
