# Location Sheet — EP13

## Primary Location: Kiko's Home — Playroom

### Description
A warm, cozy playroom in Kiko's home. Soft natural light fills the space from a large window. The room feels safe and inviting — a place where feelings can be expressed freely.

### Time of Day
- **Lighting**: Warm daytime (late morning to early afternoon)
- **Mood**: Safe, comfortable, familiar

### Key Set Pieces

| Element | Position | Description |
|---------|----------|-------------|
| Block Tower | Center-stage | Colorful building blocks stacked tall (before fall) |
| Scattered Blocks | Floor | Blocks after tower falls |
| Play Mat | Floor center | Soft, colorful mat where blocks are |
| Window | Back wall | Large, lets in warm light |
| Shelf | Back wall | More toys, books, organized |
| Small Table | Stage-left | For reference depth |
| Rug | Under mat | Soft area rug, warm tones |

### Color Palette
- Walls: Warm cream/light yellow
- Floor: Soft wood tone
- Accents: Coral, yellow, soft blue (matching characters)
- Light: Golden-warm throughout

### Lighting Notes
- Natural window light as key
- Soft fill, no harsh shadows
- Warm color temperature (~3200K feel)
- Safe, inviting atmosphere

### Camera Setup
- Two-shot framing (Kiko left, Mimi right)
- Eye-level with characters (child perspective)
- Static camera for stability
- Clean background for focus on emotions

### Sound Design
- Soft ambient room tone
- Block falling sound (gentle, not startling)
- Breathing sounds during calm moments
- Light, warm background music throughout
