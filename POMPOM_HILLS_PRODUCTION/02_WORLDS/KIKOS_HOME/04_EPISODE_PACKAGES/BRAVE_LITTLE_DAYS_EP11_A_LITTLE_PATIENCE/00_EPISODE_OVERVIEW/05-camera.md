# EP11 — Camera

## Camera Identity

| Parameter | Value |
|-----------|-------|
| Focal Length | 28–35mm range |
| Eye Level | Child eye level (approx. 80–100cm from ground) |
| Style | Warm, intimate, stable |
| Movement | Minimal — mostly static with gentle reframing |
| Aspect Ratio | 16:9 |
| Frame Rate | 24fps |

## Camera Rules

1. **Child Eye Level**: All shots at child height. No looking down on characters.
2. **Static Preferred**: Minimal camera movement. Let action fill the frame.
3. **Warm Framing**: Characters comfortably within frame, not crowded.
4. **Window Visibility**: Window with sky visible in shots 04–05, implied in others.
5. **Consistent Depth**: Same depth of field across all shots for continuity.

## Shot-by-Shot Camera Notes

### Shot 01 — Waiting for Mimi
- **Framing**: Medium shot, Mimi centred with cup, Kiko approaching from left
- **Focus**: Mimi's cup, then Kiko's reaction
- **Lighting**: Warm daylight from window right
- **Background**: Kitchen nook, cream walls

### Shot 02 — I Want It Now
- **Framing**: Two-shot — Kiko left, Mimi right with cup
- **Focus**: Kiko's impatient expression, foot tapping
- **Movement**: Static
- **Cup**: In Mimi's hands, Kiko reaching slightly

### Shot 03 — Can You Wait
- **Framing**: Wider shot to include Mimi pointing to window
- **Focus**: Mimi speaking, Kiko resisting
- **Movement**: Static — let Mimi's gesture be the motion
- **Cup**: In Mimi's hands

### Shot 04 — Looking Out the Window
- **Framing**: Medium shot, Kiko centred, window visible right
- **Focus**: Kiko's face softening, then the window
- **Movement**: Static
- **Window**: Blue sky and clouds visible

### Shot 05 — The Clouds Are Fun (Quality Reset Anchor)
- **Framing**: Wide shot, Kiko at window, clouds prominent
- **Focus**: Kiko pointing, cloud shapes
- **Movement**: Static — clouds drift slowly
- **Window**: Full cloud view with recognizable shapes
- **Quality Check**: Re-verify all character positions, colours, lighting

### Shot 06 — Almost Time
- **Framing**: Medium two-shot, both characters
- **Focus**: Mimi offering cup, Kiko's happy reaction
- **Movement**: Static
- **Cup**: Being passed from Mimi to Kiko

### Shot 07 — That Was Worth It

- **Movement**: Static

### Shot 08 — Waiting Is Okay
