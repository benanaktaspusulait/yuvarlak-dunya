# Storyboard — Kiko's Home Tour

> **15 saniye. 4 shot. Dışarıdan içeriye kesintisiz.**

## Shot 01 — Remembering (0–4 sn)

```
┌─────────────────────────────────┐
│                                 │
│    ☁️        🌳                 │
│                                 │
│         🏠 (pink house)         │
│                                 │
│    🌸  👧 → 🏠                  │
│    (remembers, turns)           │
│                                 │
│  ═══════════════════════════    │
│         stone path              │
└─────────────────────────────────┘

Camera: Wide, static, 50mm
Kiko: Smiles, turns toward house
Dialogue: "Oh! I almost forgot!"
```
