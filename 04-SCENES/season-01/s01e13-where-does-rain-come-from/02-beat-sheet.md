# S01E13 — Beat Sheet

---

## Story Rhythm

The episode keeps the same educational goal, but the answer is discovered through observation instead of explanation.

Runtime: 90 seconds.

```
Beat 1: Curiosity
Luca and Mimi climb Cloud Hill. Wind moves the grass. They notice the huge clouds.
────────────────────────────────
Beat 2: Wonder
They watch the clouds drift and change. Luca wonders if rain comes from up there.
────────────────────────────────
Beat 3: Thinking
They sit together. Mimi observes that one cloud looks full. Both watch quietly.
────────────────────────────────
Beat 4: Waiting
The hill becomes quiet. The wind moves the grass. Both children listen and look.
────────────────────────────────
Beat 5: Discovery
The first raindrop lands on Luca's hand. They notice it before naming it.
────────────────────────────────
Beat 6: Joy
They look back to the cloud as gentle rain begins. The answer feels shared.
```

---

## Cloud Progression

| Shot | Cloud State | Visual Note |
|---|---|---|
| 01 | Normal — soft white clouds | CH-CLOUD-A/B/C layout as Hero View |
| 02 | One cloud moves slowly across the sky | Same cloud identity — not a new sky |
| 03 | The same cloud looks fuller, slightly heavier | Subtle — shape thickens, not dramatic |
| 04 | The same cloud becomes slightly darker | Still pastel — not grey, not storm |
| 05 | The first small raindrop falls from that cloud | Cloud at its heaviest — still safe |
| 06 | Gentle light rain begins | Cloud releasing — soft, friendly, not scary |

The cloud progression must feel continuous — the same cloud tracked across all 6 shots. Do not generate a new sky or a different cloud in any shot.

## Dialogue Principle

Use less dialogue than before.

Let the wind, grass, clouds, silence, and raindrop carry the explanation.
