# Render Prompts — Kiko's Home Tour

> **15 saniye. 4 shot. Dışarıdan içeriye kesintisiz.**

---

## Global Style

All shots use this base style:

```
Pixar-quality preschool animation, soft plush toy texture, rounded shapes only, pastel colours, matte clay-plush surfaces, toddler-safe emotional clarity, warm diffused daylight, soft contact shadows, no hard darkness, whimsical preschool world, Pompom Hills art style, 16:9
```
