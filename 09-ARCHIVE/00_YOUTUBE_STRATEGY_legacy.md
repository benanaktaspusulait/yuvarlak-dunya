# Pompom Hills — YouTube Strategy

---

## Overview

This document defines the publishing strategy for Pompom Hills on YouTube.

The goal is to build a sustainable channel that grows organically through quality content and consistent publishing.

---

## Publishing Philosophy

- Quality over quantity
- Consistency over speed
- Evergreen content over trending content
- Audience trust over algorithm chasing

---

## Long-form Strategy

### Episode Format

| Setting | Value |
|---------|-------|
| Duration | 60 seconds (1-minute episodes) |
| Shots | 4 per episode |
| Shot duration | ~15 seconds |
| Target audience | 3–4 years old |
| Publishing frequency | 2–3 per week |

### Content Pillars

| Pillar | Description | Percentage |
|--------|-------------|------------|
| **Discovery** | New things, exploration, curiosity | 60% |
| **Comfort** | Bedtime, calm, safety | 20% |
| **Play** | Games, interaction, fun | 15% |
| **Special** | Holidays, celebrations, milestones | 5% |

### Episode Release Order

Release episodes in production order:

1. S01E01 — Merhaba Pompom Hills
2. S01E02 — Zıpla Zıpla Kiko
3. S01E03 — Küçük Çiçek
4. ... (continue through S01E26)

---

## Shorts Strategy

### Format

| Setting | Value |
|---------|-------|
| Duration | 15–30 seconds |
| Aspect ratio | 9:16 (1080×1920) |
| Content | Character intros, key moments, gentle loops |
| Publishing frequency | 3–5 per week |

### Content Types

| Type | Description | Example |
|------|-------------|---------|
| **Character Intros** | 15-second character introductions | Kiko, Mimi, Opa intros |
| **Moment Clips** | Key emotional moments from episodes | "Goodnight, stars." |
| **Gentle Loops** | Short, calming loops for background | Stars twinkling, leaves falling |
| **Discovery Clips** | Quick discovery moments | Finding a round stone |

### Shorts Best Practices

- Hook in first 2 seconds
- Clear visual focus
- No fast cuts
- Gentle music or ambient sound
- End with a calm feeling

## Playlist Strategy

### Core Playlists

| Playlist | Content | Purpose |
|----------|---------|---------|
| **Pompom Hills Full Episodes** | All 60-second episodes | Main content |
| **Character Intros** | All character introduction shorts | Discovery |
| **Bedtime Stories** | Calm, sleepy episodes | Evening viewing |
| **Nature & Discovery** | Episodes about nature | Educational |
| **Best Of** | Most popular episodes | New viewer entry point |
| **New Episodes** | Latest uploads | Subscriber retention |

### Playlist Rules

- Each playlist should have at least 5 videos
- Playlists should be updated when new content is added
- "Best Of" playlist should be refreshed monthly

---

## Evergreen Content

Pompom Hills content should be evergreen — relevant at any time, not tied to trends.

### Evergreen Topics

- Bedtime routines
- Nature discovery
- Friendship
- Sharing
- Colours and shapes
- Feelings and emotions
- Seasonal themes (spring, summer, autumn, winter)

### Avoid

- Trending topics that will date quickly
- Current events references
- Pop culture references
- Technology-focused content

## Character Introduction Strategy

### Release Order

Release character intros in this order:

1. Kiko (main character — first impression)
2. Mimi (best friend — immediate companion)
3. Opa (guide — safety figure)
4. Luca (explorer — variety)
5. Noah (builder — creativity)

### Intro Publishing

- Also release as a regular video
- Create compilation videos (2s, 3s, all)
- Use intros as channel trailer

---

## Season Launch Strategy

### Pre-Launch

- Release character intros (2 weeks before)
- Release teaser clips
- Create channel trailer from intros

### Launch Week

- Release S01E01 on launch day
- Release S01E02 two days later
- Release S01E03 four days later
- Post daily Shorts during launch week

### Post-Launch

- Maintain 2–3 episodes per week
- Release 3–5 Shorts per week
- Engage with comments (if enabled for older audience)
- Monitor analytics and adjust

## Metadata Philosophy

### Title Principles

- Clear and descriptive
- Character name included
- Action or discovery mentioned
- Emoji for visual appeal
- "Pompom Hills" at the end
- Maximum 60 characters

### Description Principles

- First 3 sentences: episode summary
- Middle: learning points and characters
- End: call to action (subscribe, like)
- Keywords for SEO

### Tag Principles

- 15–20 tags per video
- Mix of broad and specific terms
- Include: preschool, toddler, gentle, safe, educational
- Include: character names
- Include: episode-specific terms
