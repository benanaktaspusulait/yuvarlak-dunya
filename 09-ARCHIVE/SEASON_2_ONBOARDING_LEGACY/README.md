# Season 2 — Onboarding Package

---

## Purpose

This folder contains all onboarding materials for new Season 2 characters and worlds.

Following the Character and World Onboarding Policy (`11-DOCS/02_SEASON_STRATEGY.md`), no new major character or reusable world enters the regular episode system without an onboarding video first.

---

## Policy Reference

| Rule | Detail |
|------|--------|
| Core Rule | Onboarding video required before regular use |
| Character Cadence | 1 new character every 2–3 months |
| World Rollout | Only when story needs them and pipeline supports them |
| Onboarding Order | Canon docs → Hero View → Intro video → Shorts → First story → Regular use |

## Onboarding Checklist

For each character:

- [ ] Character bible complete
- [ ] Character reference approved
- [ ] Intro video script written
- [ ] Intro video produced (30–60s)
- [ ] 3–5 Shorts extracted
- [ ] Character caption/description finalized
- [ ] First simple story episode written
- [ ] First story produced
- [ ] Regular episode use begins

For each world:

- [ ] World bible complete
- [ ] World spec complete
- [ ] Hero View approved
- [ ] Intro video script written
- [ ] Intro video produced (30–90s)
- [ ] 3–5 Shorts extracted
- [ ] First story episode using world written
- [ ] First story produced
- [ ] Regular episode use begins
