# Pompom Hills — OpenArt World Prompt Pack

## Usage

Copy-paste these prompts directly into OpenArt. Each prompt is self-contained and includes the full style foundation. For best results, generate the valley overview first, then use it as a reference image for subsequent generations.

4. **Kiko's house interior** — establish the living space
5. **Mimi's burrow** — establish the companion's space
6. **Opa's tree** — establish the guide's space
7. **Individual hills** — build the geography
8. **Sky conditions** — establish time of day
9. **Trio shot** — bring characters into the world
