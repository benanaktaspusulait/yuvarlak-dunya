# Mekan Optimizasyonu — Gece-Gündüz Döngüsü

## 1. Amaç

34 mekanın "Gece / Işıkları Sönmüş" versiyonunu oluşturarak:
- Goodnight Pompom Hills serisi için kullanılacak
- Prodüksiyon maliyeti artırılmadan yeni mekan hissi yaratılacak
- Çocukların gece rutinini görsel olarak desteklenecek

---

## 2. Seçilen 5 Ana Mekan

| Mekan | Gündüz | Gece Değişiklikleri |
|-------|--------|---------------------|
| **Central Square** | Açık, güneşli | Koyu lacivert gökyüzü, sarı pencere ışıkları |
| **Opa's Tree** | Yeşil, sakin | Loş iç mekan, sıcak lamba ışığı |
| **Kiko's Home** | Sıcak, davetkar | Perdeler kapalı, yatak ışığı |
| **Mimi's Burrow** | Pofuduk, rahat | Koyu mavi duvarlar, ay ışığı |
| **Flower Hill** | Renkli, canlı | Soluk renkler, yıldızlar |

## 4. Overlay Katmanları

### Yıldız Katmanı
- Sarı, küçük daireler
- Rastgele dağılım
- Yavaşça parıldama efekti

### Işık Katmanı
- Pencere ışıkları (sıcak turuncu)
- İç mekan lamba (sarı)
- Ay ışığı (beyaz-mavi)

### Atmosfer Katmanı
- Genel karartma (%30 opacity)
- Sıcak tonlama (turuncu overlay)
