# QC Checklist — PomPom Hills

---

## Amaç

Her bölüm yayın öncesi bu kontrol listesinden geçirilir.

---

## 1. Görsel Tutarlılık Kontrolü

### Karakter Kontrolü
- [ ] Karakter renkleri doğru mu? (Kiko #F8BBD0, Mimi #90CAF9, Opa #A5D6A7)
- [ ] Karakter kıyafetleri doğru mu? (Kiko: coral shirt + white shorts, Mimi: yellow t-shirt, Opa: brown shawl)
- [ ] Karakter ölçekleri doğru mu? (Kiko=100, Mimi=80, Opa=120)
- [ ] Karakter yüzleri tanımlı mı? (Büyük gözler, küçük ağız, yuvarlak yüz)
- [ ] Karakter ifadeleri okunabilir mi? (1 saniyede anlaşılır)
- [ ] Karakter elleri/parmakları doğru mu? (Mitten/paw/kanat ucu, parmak yok)

### Mekan Kontrolü
- [ ] Mekan değişmiş mi? (LOCKED olmalı)
- [ ] Giant Pompom Tree aynı mı? (#81C784 taç, şekil sabit)
- [ ] Mimari değişmiş mi? (Evler, kapılar, pencereler aynı)
- [ ] Yollar aynı mı? (Yuvarlak taşlar, kavisli patika)
- [ ] Çiçekler aynı mı? (Renk ve yerleşim sabit)
- [ ] Banklar/lamba direkleri aynı mı?

### Işık Kontrolü
- [ ] Işık yönü tutarlı mı?
- [ ] Işık rengi tutarlı mı? (Sıcak günışığı)
- [ ] Gölge opaklığı doğru mu? (%25 altında)
- [ ] Gece sahnelerinde siyah yok mu? (Yumuşak mavi)
- [ ] Karakter yüzünde gölge var mı? (Olmamalı)

### Renk Kontrolü
- [ ] Renkler pastel mi? (Neon yok)
- [ ] Maksimum 3 baskın renk mi?
- [ ] Beyaz alan bırakılmış mı?
- [ ] Karakterler arka plandan ayrılıyor mu?

- [ ] Diyaloglar net mi?

- [ ] Müzik tanıdık mı?
- [ ] SFX yeterince yumuşak mı?
- [ ] Doğa sesleri doğru mu? (Kuş, rüzgar, su)
