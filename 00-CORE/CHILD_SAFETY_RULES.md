# Child Safety Rules — PomPom Hills

---

## Temel Kural

> **"3-4 yaş çocuk bunu 2 saniyede okuyabilir, korkmaz, kimin ne hissettiğini anlar."**

---

## 1. Yasak İçerik Listesi

### 1.1 Görsel Yasaklar

```
✗ Sivri köşe, sivri uç, keskin kenar
✗ Karanlık, korku, gotik atmosfer
✗ Gerçekçi/ürkütücü yüz ifadeleri
✗ Kan, yaralanma, morluk
✗ Silah, bıçak, kalkan, ok
✗ Yangın, patlama, duman
✗ Çöp, kirlilik, bozulma
✗ Çukur, uçurum, düşme riski
✗ Hızlı hareket, sarsıntılı kamera
✗ Neon, koyu doygun renkler
✗ Karanlık gökyüzü (siyah gece yok)
✗ Gerçekçi insan yüzleri
✗ Büyük, korkutucu gölgeler
```

### 1.2 Ses Yasakları

```
✗ Bağırmak, çığlık atmak
✗ Korkutucu fısıltı
✗ Sert gürültü, patlama sesi
✗ Motor, siren, alarm
✗ Agresif müzik, hızlı ritim
✗ Ani ses değişimi
✗ Yüksek desibel ses (>85 dB)
✗ Kırılma, düşme, çarpma sesi
✗ Uğursuz/gergin müzik
```

### 1.3 Davranışsal Yasaklar

```
✗ Kavga, dövüş, itiş
✗ Cezalandırma, azarlama
✗ Alay, küçümseme, dışlama
✗ Yalnız bırakma, terk etme
✗ Korkutma, tehdit
✗ Yalan söyleme, kandırma
✗ Kıskançlık, rekabet
✗ Öfke, kızgınlık
✗ Tehlikeli davranış (yükseğe çıkma, koşma)
✗ Yetişkin davranışı (parmak sallama, göz devirme)
```

---

## 2. İzin Verilen İçerikler

### 2.1 Görsel İzinler

```
✓ Yuvarlak, yumuşak şekiller
✓ Pastel, sıcak renkler
✓ Yumuşak ışık, gölge yok denecek kadar az
✓ Doğal ortam (çiçek, ağaç, gökyüzü)
✓ Basit, okunabilir kompozisyon
✓ Tek odak noktası
✓ Bol negatif alan
```

### 2.2 Ses İzinleri

```
✓ Yumuşak, sıcak diyalog
✓ Akustik enstrümanlar (marimba, piyano, ukulele)
✓ Doğa sesleri (kuş, rüzgar, su)
✓ Yumuşak SFX ("boop", "pop", "ting")
✓ Yavaş tempo (60-120 BPM)
✓ Ninni, rahatlatıcı melodi
```

### 2.3 Davranış İzinleri

```
✓ Merak, keşif, öğrenme
✓ Paylaşım, yardımlaşma
✓ Sabır, bekleme
✓ Naziklik, sevgi
✓ Güven, huzur
✓ Uyku, dinlenme
✓ Kutlama, neşe
✓ Doğaya saygı
```

---

## 3. Ses Desibel Limitleri

| Ses Tipi | Maks. Desibel | Not |
|----------|---------------|-----|
| Diyalog | -6 dB | Ana ses |
| Müzik | -12 dB | Arka plan |
| SFX | -10 dB | Tamamlayıcı |
| Ambiyans | -18 dB | Çok sessiz |
| **Toplam** | **-3 dB** | **Asla 0 dB'yi geçme** |

### Ani Ses Değişim Kuralları

- Fark: Maks. 6 dB (2 saniye içinde)
- Yükseliş: Yavaş, kademeli
- Düşüş: Yavaş, kademeli
- Kesme: Asla ani kesme

## 5. Kültürel Hassasiyet

### 5.1 Yasak Öğeler

```
✗ Dini semboller (haç, hilal, yıldız, tapınak)
✗ Siyasi semboller (bayrak, amblem)
✗ Silah benzeri nesneler (kılıç, ok, tabanca)
✗ Tütün/alkol ürünleri
✗ Kumar nesneleri (zar, kart)
✗ Gerçekçi yaralanma/hastalık
✗ Irk/dil/cinsiyet ayrımcılığı
✗ Yetişkin içerikleri
```

### 5.2 İzin Verilen Öğeler

```
✓ Doğal ortamlar (orman, bahçe, göl)
✓ Evrensel semboller (kalp, yıldız, güneş)
✓ Hayvanlar (tavşan, baykuş, kedi, kuş)
✓ Yiyecekler (meyve, sebze, ekmek)
✓ Oyuncaklar (top, puzzle, kitap)
✓ Müzik aletleri (marimba, piyano)
✓ Doğa olayları (yağmur, kar, rüzgar)
```

---

## 6. Karakter Davranış Kuralları

### 6.1 Her Karakter İçin

- Nazik, sıcak, sakin hareket
- Yüz ifadesi 1 saniyede okunabilir
- Ses yumuşak, bağırmak yok
- Arkadaşlarına yardım eder
- Doğaya saygı gösterir
- Acele etmez, sabırlıdır

### 6.2 Karakter Etkileşim Kuralları

```
✓ Birbirine selam verir
✓ Birbirine yardım eder
✓ Birlikte keşfeder
✓ Birlikte oynar
✓ Birlikte öğrenir
✗ Birbirini itmez
✗ Birbirini alay etmez
✗ Birbirini dışlamaz
✗ Birbirini korkutmaz
```
