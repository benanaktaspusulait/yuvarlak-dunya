# Audio Style Guide — Pompom Hills v2.1

---

## Genel Ses Felsefesi

Pompom Hills'in sesi, dünyadaki gibi yumuşak, sıcak ve güvende olmalıdır.

Çocuklar seslerle rahatlamalı, korkmamalı, heyecanlanmamalı.

Her ses "sarılma" hissi vermeli.

---

## Voice Identity

Aynı karakter her shot'ta aynı ses kimliğini korumalıdır.

Maintain:

- same voice identity
- same age impression
- same pitch
- same timbre
- same speaking speed
- same warmth
- same preschool energy
- same speaking rhythm
- same pronunciation
- same accent
- same recording quality

Voice Reference veya Voice ID destekleyen sistemlerde aynı karakter için her shot'ta aynı Voice ID kullanılmalıdır.

Bir bölümdeki tüm speaking shot'lar aynı kayıt seansında kaydedilmiş gibi duyulmalıdır.

Önceki shot mevcutsa, önceki shot aynı zamanda voice reference olarak kabul edilir.

Farklı narrator, alternate voice, daha büyük/küçük yaş izlenimi, farklı accent, farklı pronunciation veya farklı performer hissi kabul edilmez.

## Müzik Stili

### Tempo

| Durum | Tempo | Not |
|---|---|---|
| Normal sahne | 80-100 BPM | Yavaş, rahat |
| Mutlu an | 100-120 BPM | Hafif enerjik |
| Uyku sahnesi | 60-70 BPM | Çok yavaş, ninni |
| Keşif anı | 90-110 BPM | Meraklı, hafif |

### Enstrümanlar

| Enstrüman | Kullanım | Not |
|---|---|---|
| Marimba | Ana melodi | Yuvarlak, sıcak, oyuncak hissi |
| Soft Piyano | Yumuşak anlar | Sıcak, basit notalar |
| Glockenspiel | Sihirli anlar | Parıltı, kelebek, yıldız |
| Ukulele | Neşeli anlar | Hafif, ritmik |
| Kslofon | Çocuk temalı | basit, tanıdık |
| Yumuşak tef | Ritim | Hafif, yumuşak |

### Yasak Enstrümanlar

```
✗ Electric guitar
✗ Drums (sert)
✗ Synthesizer (sert)
✗ Orchestra (büyük, dramatik)
✗ Bass (derin, karanlık)
✗ Horn (sert, yüksek)
```

## Sahne Bazlı Ses Örnekleri

### Sabah Sahnesi

```
Ambiyans: Kuş cıvıltısı + hafif rüzgar
Müzik: Marimba + soft piyano, 90 BPM
SFX: Kapı "squeak", adım sesleri
```

### Oyun Sahnesi

```
Ambiyans: Kuş cıvıltısı
Müzik: Ukulele + marimba, 110 BPM
SFX: "Boop" (top), gülme, koşma
```

### Keşif Sahnesi

```
Ambiyans: Rüzgar + uzak kuş
Müzik: Glockenspiel + piyano, 100 BPM
SFX: "Ting" (bulma), "pop" (merak)
```

### Uyku Sahnesi

```
Ambiyans: Çok sessiz, hafif rüzgar
Müzik: Soft piyano ninni, 65 BPM
SFX: Yumuşak nefes, esneme
```

### Yağmur Sahnesi

```
Ambiyans: Yumuşak yağmur + uzak gök gürültüsü (çok hafif)
Müzik: Piyano + marimba, 80 BPM
SFX: Yağmur damlaları (yuvarlak ses), kapı kapanma
```

## Müzik Yapısı

### Bölüm Başlangıcı

- Kısa melodi (5-10 saniye)
- Tanıdık ritim
- "Bu bizim dünyamız" hissi

### Sahne Geçişleri

- Yumuşak crossfade (3-5 saniye)
- Ani kesme yok
- Melodi devamlılığı

### Bölüm Kapanışı

- Aynı melodi (başlangıçla aynı)
- Yavaşlayarak bitme
- "İyi geceler" hissi
