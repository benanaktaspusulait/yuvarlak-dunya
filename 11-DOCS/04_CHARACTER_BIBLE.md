## Main Trio

### Kiko

| Attribute | Detail |
|-----------|--------|
| **Role** | Main character — the child audience identifies with |
| **Species** | Human girl |
| **Age feel** | 4 years old |
| **Personality** | Curious, gentle, kind, brave |
| **Emotional strengths** | Asks questions, explores with kindness, includes everyone |
| **Weaknesses** | Sometimes hesitant, needs encouragement |
| **Relationships** | Mimi (best friend), Opa (mentor), Luca (discovery partner) |
| **Typical story role** | Leads most episodes, drives discovery |
| **Speech style** | "What's that?" / "Look!" / "Let's see!" |
| **Visual identity** | Coral pink shirt, white shorts, brown pigtails |
| **Animation notes** | Gentle bounce when walking, pigtails sway |
| **Production status** | Production Ready |

## Character Rules

- Characters are permanent — never retired or replaced
- Appearance never changes across episodes
- Maximum 2–3 active characters per episode
- New characters introduced only after core trio established
- No more than one new character per 5-episode block
- Characters never fight, tease, or exclude each other
