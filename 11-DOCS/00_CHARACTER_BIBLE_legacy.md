# Pompom Hills — Character Bible

---

## Overview

Pompom Hills features a small cast of memorable characters designed for children aged 3–4.

Each character has a clear role, distinct personality, and unique visual identity.

Characters are permanent assets — they never change, retire, or get replaced.

---

## Character Rules

- Characters are permanent assets — never retired or replaced
- Character appearance never changes across episodes
- Maximum 2–3 active characters per episode
- New characters introduced only after the core trio is established
- No more than one new character per 5-episode block
- Characters never fight, tease, or exclude each other
