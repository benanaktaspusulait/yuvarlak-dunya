# Brand Bible

> *Source of truth for: Brand identity, voice, visual rules, licensing direction.*
> *Every piece of external communication should reference this document.*

---

## Brand Mission

Pompom Hills exists to give children a world where they feel safe, curious, and loved.

Every frame, every sound, every word serves this mission. The brand is not a product — it is a promise: *this is a safe place for your child.*

---

## Brand Vision

To become the world's most trusted preschool animation brand — known for gentleness, quality, and emotional honesty.

Pompom Hills does not chase trends. It builds trust. Over years, not weeks.

---

## Brand Values

| Value | Meaning |
|-------|---------|
| **Gentleness** | Every interaction is soft, warm, and safe |
| **Curiosity** | Discovery is celebrated, never forced |
| **Honesty** | Emotions are real, never performative |
| **Consistency** | The world never changes unexpectedly |
| **Quality** | Every frame meets the same standard |
| **Inclusion** | Every child sees themselves in this world |

---

## Brand Personality

If Pompom Hills were a person, it would be:

- Warm, never cold
- Patient, never rushed
- Gentle, never loud
- Honest, never performative
- Simple, never complicated
- Safe, never risky

Pompom Hills speaks like a kind grandparent: slow, warm, and full of quiet love.

## Brand Voice

### How We Speak

| Do | Don't |
|----|-------|
| Use simple, warm language | Use complex or technical language |
| Speak directly to children | Talk over children's heads |
| Be calm and reassuring | Be excited or frantic |
| Celebrate small moments | Chase big spectacles |
| Use repetition for learning | Assume children remember |

### Tone Variations

| Context | Tone |
|---------|------|
| YouTube titles | Warm, clear, inviting |
| Social media captions | Friendly, simple, encouraging |
| Merchandise packaging | Playful, gentle, trustworthy |
| Licensing communications | Professional, warm, brand-aligned |
| Parent-facing content | Honest, reassuring, quality-focused |

### Voice Examples

**YouTube Title:**
"Kiko Discovers a Round Stone 🪨 | PomPom Hills"

**Social Caption:**
"Kiko found something round today. Can you guess what it is? 🌿"

**Merch Tagline:**
"Gentle adventures for little explorers."

## Logo Usage

### Primary Logo

- Always use the official Pompom Hills logo
- Never recreate, redraw, or modify the logo
- Minimum clear space: logo height on all sides
- Never place on busy backgrounds without sufficient contrast

### Logo Colours

- Primary: Coral pink (#F8BBD0)
- Secondary: Soft blue (#90CAF9)
- On light backgrounds: use primary version
- On dark backgrounds: use reversed version (if available)

### Logo Don'ts

- Never stretch or distort
- Never rotate
- Never add effects (shadow, glow, outline)
- Never place on competing patterns
- Never use low-resolution versions

---

## Typography

### Primary Typeface

- Round, friendly, sans-serif
- Examples: Nunito, Quicksand, Poppins (rounded variants)
- Never: serif, angular, decorative, or handwritten fonts

### Type Hierarchy

| Use | Weight | Size |
|-----|--------|------|
| Headlines | Bold | Large |
| Subheadings | Medium | Medium |
| Body text | Regular | Small |
| Captions | Light | Smallest |

### Typography Rules

- Always use sentence case (not ALL CAPS)
- Always use high contrast against background
- Never use text over busy video frames
- Always test readability at small sizes

## Thumbnail Identity

### Structure

Every thumbnail follows this layout:

```
[ Character face ] + [ Object of discovery ] + [ Warm background ]
```

### Rules

- 1280×720 minimum resolution
- One happy character face (always visible)
- Clear, simple action or object
- Warm, pastel background
- No text over character face
- Maximum 4 words of text (if any)
- Consistent style across all thumbnails

### Thumbnail Checklist

- [ ] Character face visible and large
- [ ] Warm, inviting colours
- [ ] Clear focal point
- [ ] No cluttered elements
- [ ] Text readable at small size
- [ ] Consistent with brand palette

## Social Media Identity

### Platform Strategy

| Platform | Content Type | Frequency | Tone |
|----------|-------------|-----------|------|
| YouTube | Episodes + Shorts | 2-3/week + 3-5/week | Warm, inviting |
| Facebook | Stories + Episodes | Daily + weekly | Friendly, shareable |
| Instagram | Reels + Stills | 3-5/week | Beautiful, calm |
| TikTok | Shorts | 3-5/week | Fun, gentle |

### Social Media Rules

- Never use trending audio that is not brand-safe
- Never engage in controversy
- Never respond to negative comments publicly
- Always maintain the gentle tone
- Always use brand colours and fonts
- Never post content that is not reviewed

## Licensing Direction

### Target Markets

| Market | Priority | Notes |
|--------|----------|-------|
| United Kingdom | Primary | Base market, licensing hub |
| Europe (EU) | Secondary | Via UK licensing events |
| English-speaking markets | Tertiary | US, Canada, Australia |
| Middle East | Future | Via local licensing partners |

### Licensing Categories

| Category | Priority | Notes |
|----------|----------|-------|
| Publishing (books) | High | Natural extension |
| Plush toys | High | Core merchandise |
| Apparel | Medium | Children's clothing |
| Stationery | Medium | School supplies |
| Digital (apps) | Future | Educational games |
| Home (bedding, decor) | Future | Bedroom merchandise |

### Licensing Rules

- All licensees must maintain brand quality standards
- All products must be reviewed before production
- All products must meet child safety standards
- No product may alter character design
- All packaging must use approved brand assets

## Brand Do's and Don'ts

### Do

- Maintain warmth in every communication
- Use simple, clear language
- Celebrate small discoveries
- Keep visuals soft, round, and pastel
- Treat every audience interaction as a trust-building moment

### Don't

- Use aggressive marketing language
- Chase trends that compromise brand identity
- Create content that overstimulates children
- Alter character designs for commercial purposes
- Engage in competitive or negative messaging

---

## Quality Standards

Every piece of content that represents Pompom Hills must pass this test:

> **Would a parent feel safe showing this to their 3-year-old?**

If the answer is not an immediate "yes," the content is not ready.
